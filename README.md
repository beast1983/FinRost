# ФинРост (FinRost)

**Финансовый рост** — приложение для учёта инвестиций: депозитные счета, активы (акции, облигации, фонды, крипто), транзакции, срезы портфеля и аналитика.

## Возможности

- **Депозитные счета** — учёт счетов в разных валютах (RUB, USD, EUR, CNY), пополнения и списания.
- **Активы** — покупка, докупка, продажа акций, облигаций, фондов; автоматический пересчёт средней цены.
- **Транзакции** — журнал всех операций (покупки, продажи, купоны, дивиденды, пополнения, списания).
- **Срезы портфеля** — ежемесячные снимки стоимости по каждому счёту.
- **Анализ** — динамика портфеля, доходность, доходы и эффективность с графиками.
- **Курсы валют** — автоматическое обновление с ЦБ РФ.
- **Импорт** — исторические срезы и доходы из CSV.
- **Реестр тикеров** — справочник тикеров с названиями.
- **Архивация** — автоматическое резервное копирование базы данных в ZIP при закрытии программы.

## Технологии

- Python 3.10+
- tkinter + ttkbootstrap (интерфейс)
- SQLite (база данных)
- matplotlib (графики)
- requests (курсы ЦБ РФ, цены Московской биржи)

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

При первом запуске создаётся база данных `investments.db` рядом с приложением. При обновлении версии схемы миграции применяются автоматически (см. `db_schema.py`).

## Структура проекта

- `main.py` — точка входа, главное окно
- `version.py` — название и версия приложения
- `database.py` — слой данных (SQLite): запросы к таблицам
- `db_schema.py` — создание схемы БД и миграции (первый запуск / обновление версии)
- `app_config.py` — конфигурация (путь к БД, настройки архивации)
- `accounts_view.py` — вкладка депозитных счетов
- `assets_view.py` — вкладка активов
- `transactions_view.py` — вкладка транзакций
- `cuts_view.py` — вкладка срезов
- `analysis_view.py` — вкладка анализа
- `settings_view.py` — настройки (валюты, импорт, база данных, архивация, тикеры)
- `api_client.py` — внешние API (ЦБ РФ, Московская биржа)
- `snapshot_guard.py` — проверка актуальности данных и сохранение срезов
- `calendar_utils.py`, `table_utils.py` — вспомогательные утилиты
