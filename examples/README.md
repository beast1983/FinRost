# Примеры файлов импорта

Папка содержит примеры CSV-файлов для импорта данных в программу.
Раздел → **Настройки → Общие → Импорт**.

## Общие правила

- **Разделитель** — точка с запятой (`;`)
- **Кодировка** — UTF-8 (желательно с BOM). Также поддерживаются UTF-16 и cp1251/cp866
- **Десятичный разделитель** — запятая (`320,50`)
- **Разделитель тысяч** — пробел или без (`1 234,56`)
- **Пустая ячейка** = пропуск (нет значения за этот месяц)
- Первая строка — заголовок (при импорте пропускается)

## Файлы

### assets_example.csv — стоимость активов

Шапка: `Актив;Тикер;Тип;Валюта;` + 12 колонок месяцев (`Янв;Фев;...;Дек`)

Колонка **Тип** определяет характер строки:

| Тип         | Назначение                                                  |
|-------------|-------------------------------------------------------------|
| `акция`     | Акция                                                       |
| `облигация` | Облигация                                                   |
| `фонд`/`etf`/`пиф`/`зпиф` | Паевой фонд / ETF                              |
| `баланс`    | Остаток на счёте за месяц (специальная строка)              |
| `пополнил`  | Пополнения и выводы: плюс = внесено, минус = выведено       |

- Если **Тикер** пустой — программа создаст условный (`АКТИВ_1`, `АКТИВ_2`…).
- Колонка **Валюта** — код валюты: `RUB`, `USD`, `EUR`, `CNY`.

### incomes_example.csv — доходы (купоны и дивиденды)

Та же шапка, что у активов. В колонке **Тип** — только `купон` или `дивиденд`.

### tickers_example.csv — реестр тикеров

Две колонки: `Тикер;Название`. Используется для заполнения справочника тикеров.

## Как попробовать

1. Вкладка **Настройки → Тикеры** → «⬆ Импорт» → выбрать `tickers_example.csv`
2. Вкладка **Настройки → Общие → Импорт → Стоимость активов**:
   - выбрать брокера и год;
   - выбрать `assets_example.csv` → «Импорт срез»
3. Аналогично для доходов — `incomes_example.csv` → «Импорт доходов»
